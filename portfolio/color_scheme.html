Color Scheme Generator

Create and save awesome palettes

This App enables users to search for color schemes based on a selected color. It includes several alternatives for the palette type resulting and allows users to save their preferred color schemes to be used later.

I love building this type of project where it becomes a useful tool for myself and others.
