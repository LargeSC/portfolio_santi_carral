Message in a blockchain

A messaging app on Ethereum test-net

This project is one of the coolest I've done, it allows users to send me messages and store them in the blockchain. I built the project following a Buildspace course using React, Solidity & Hardhat to build the app.

View Live View Code

Some of my biggest learnings and achievements for this project were:

  • Writing a Smart Contract using Solidity.
  • Compiling and deploying the contract using HardHat & Alchemy.
  • Adding additional projects to the original project like the ability to send messages and display them.
  • Developing front-end for DAPP to connect wallet, send messages and display previous messages.

I am very excited with Web 3's decentralization and how concepts like DAOs will revolutionize community collaboration.
