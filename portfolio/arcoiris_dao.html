Arcoiris DAO

A DAO on Ethereum test-net

This is a project that I worked on with Buildspace & Thirdweb. The goal was to create a DAO on Ethereum's Rinkeby test network.

View Live View Code

Some of my biggest learnings and achievements for this project were:

  • Deploying my 1st NFT ERC-1155 contract.
  • Customizing a Smart Contract using ThirdWeb's tools.
  • Deploying an ERC-20 token, so basically my own crypto-currency.
  • Assigning tokens to the DAO contract and developing 100% automatic voting rules.
  • Developing front-end for DAPP to connect wallet and perform actions.

I am very excited with Web 3's decentralization and how concepts like DAOs will revolutionize community collaboration.
